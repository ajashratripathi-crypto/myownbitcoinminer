CYD Bitcoin Miner

Install the Bitcoin hashing dashboard directly from Chrome. No Arduino IDE or Linux tools required.

ESP32-2432S028R · 2.8-inch CYD
Use Chrome or Microsoft Edge on desktop.
1. Connect the CYD using a USB data cable.
2. Turn Linux USB sharing off.
3. Click Install Bitcoin Miner.
4. Select USB Serial.
5. Wait for installation to finish.
Installing firmware will replace the program currently on the CYD.
